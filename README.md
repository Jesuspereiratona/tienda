# Mi Tiendita 

Proyecto base de una tienda en Node.js con Express y Handlebars.

## Repositorio

https://github.com/Jesuspereiratona/tienda

## Qué se hizo

Se agregaron los paquetes chalk y dayjs para mostrar mensajes con colores y fecha actual en la consola al iniciar el servidor.

## Cómo correrlo

Instalar dependencias:
npm install

Iniciar el servidor:
npm start

## Stack

- Node.js
- Express
- Handlebars
- Chalk
- Dayjs
